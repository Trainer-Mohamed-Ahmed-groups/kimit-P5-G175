1- <title></title> => tag is used to define the title of the HTML document. The text within this tag typically appears in the browser's title bar or tab.

2- <meta charset="UTF-8"> => Specifies the character set encoding for the document. This tag ensures that special characters are displayed correctly.

3- <meta name="description" content=""> => Provides a concise description of the page content. While it doesn't directly influence search rankings, it is often used as the snippet displayed in search results, so it should be compelling and relevant.

4- <meta name="keywords" content=""> => Historically, search engines used this tag to identify keywords for a page. However, modern search engines primarily rely on content analysis, and the <meta name="keywords"> tag is less significant. Some SEO experts argue that it's still a good practice to include relevant keywords.

5- <meta name="robots"> => This tag provides instructions to web crawlers on how to index or not index a page. For example, you can use it to prevent search engines from indexing certain parts of your site.

6- <link rel="canonical"> => Specifies the preferred version of a URL, especially in cases where duplicate content might exist. It helps search engines understand which version of a page should be considered authoritative.

7- <meta name="viewport"> => While not directly related to SEO, setting the viewport for mobile devices is important for user experience. Search engines consider mobile-friendliness as a ranking factor.
